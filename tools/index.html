Battle for the Net: Developer tools

Please excuse our look while this page is under construction.



We are working on developer tools to allow you to join the Battle for the Net with your site. If you use any of these, please reach out to Jeff Lyon <jeff@fightforthefuture.org> so you can stay appraised of technical developments.

Modal/Banner Widget Code

We are working on a widget that anyone can easily embed in their web site with just one line of Javascript code. This will be customizable to either show a modal action form or a banner. In both cases, some kind of "loading" animation will be displayed to the user.


A mockup of how the banner will look

We've already done this on a past net neutrality action and our working to customize for September 9th.

  • See demo of previous widget here.
    Note: the final widget for our September net neutrality action will be completely different, this is just an example of what we are capable of. We will not block page loading for the final version.
  • Github project for widget lives here.
    The final version will be in the same project and have a different look and customization options, so check back soon for more updates!

This project is open source so you will be to use ours or customize yours as needed. In either case we will be glad to assist with integration and customization.

Embeddable action form

We also have an action form you can embed on your site. This is already working and used in Namecheap's netneutrality.com action page. 

<iframe src="https://www.battleforthenet.com/embed.html"
    frameborder="0"
    scrolling="no"
    allowTransparency="true"
    style="width: 100%; height: 480px;"></iframe>
(note this may change before Sept 9th, please contact <jeff@fightforthefuture.org> so we can keep you up to date)
